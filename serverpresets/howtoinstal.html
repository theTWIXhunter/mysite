How to Install Minecraft Server Presets

A step-by-step guide to help you get your server up and running quickly!

Installation Instructions

Follow these steps carefully to install your Minecraft server preset:

  1. Access your File Manager: Log in to your hosting panel and navigate to the File Manager section where your server files are stored.
  2. Delete existing files: Select all files in the File Manager and delete them to ensure you start with a clean slate. Be careful not to delete files unintentionally!
  3. Download the server preset: Visit our website, locate the server preset you wish to install, and download the ZIP file to your computer.
  4. Upload the ZIP file: In your File Manager, click on the upload option and select the ZIP file you just downloaded. Wait until the upload completes.
  5. Unarchive the ZIP file: Once the ZIP file is uploaded, use the "Extract" or "Unarchive" option in your File Manager to unpack the contents of the ZIP file.
  6. Start your server: After the files are unarchived, return to the control panel and start your server. Your preset should now be fully installed and ready to use!
  7. Test your server: Join your server using your Minecraft client to ensure everything is working as expected. If you encounter issues, check the console logs or reach out for support.

And that's it! You now have your Minecraft server set up with the preset of your choice. Enjoy building, surviving, and adventuring!
