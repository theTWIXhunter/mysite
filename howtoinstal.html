Installation Instructions

Follow these steps carefully to install your Minecraft server preset:

  1. Access your File Manager: Log in to your hosting panel and navigate to the File Manager (the section where your server files are stored.)
  2. Delete existing files: Select all files in the File Manager and delete them to ensure you start with a clean state. Be careful not to delete files unintentionally!

    * You can leave the world files if these are not included in the preset

    * Make sure to empty the recycle Bin to avoid wasting your diskspace

  3. Download the server preset: Visit our website, locate the server preset you wish to install, and download the ZIP file to your computer.
  4. Upload the ZIP file: In your File Manager, click on the upload option and select the ZIP file you just downloaded. Wait until the upload completes.
  5. Unarchive the ZIP file: Once the ZIP file is uploaded, use the "Extract" or "Unarchive" option in your File Manager to unpack the contents of the ZIP file.
  6. Delete the ZIP file: You can delete the ZIP file once it is fully extracted.
  7. Start your server: After the files are unarchived, return to the servers console and (re)start your server. Your preset should now be fully installed and ready to use!
  8. Test your server: Join your server using your Minecraft client to ensure everything is working as expected.

    * You might need some client modifications to join if specified on the preset download page.

And that's it! You now have your Minecraft server set up with the preset of your choice. Enjoy building, surviving, and adventuring!
